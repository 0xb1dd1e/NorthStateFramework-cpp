Detailed Description

template<class ObjectType, class ContextType>
class NorthStateFramework::NSFVoidMemberAction< ObjectType, ContextType >

Represents a member function action.

Template Parameters
ObjectTypeThe class type containing the member function.
ContextTypeThe context type passed as an argument to the member function.

An action is a delegate type used by the North State Framework. This class can be instantiated with either a unary or nullary (argument-less) member function. This allows nullary delegates to be placed in lists of unary delegates and called without arguments via the function operator. This feature is convenient for attaching existing nullary methods to action lists, without having to wrap them in action methods.
