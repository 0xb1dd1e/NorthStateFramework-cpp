Detailed Description

Represents an operating system mutex that may be used to provide mutual exclusion for a shared resource.

Creates an operating system mutex.

This is the factory method for creating a mutex specific to the OS. A mutex is unlocked at creation.

virtual void lock ( )

Locks the mutex.

If the mutex is already locked when the calling thread calls lock, then the calling thread will block until the mutex is unlocked.
